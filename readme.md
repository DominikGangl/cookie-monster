# Readme
This is an educational game on dark patterns in cookie banners by David Leimstädtner and Dominik Gangl.

## Installing and running the project
To install the app, you need Node version 18.13+ (Node 20 LTS recommended). 

## Start Game
To start the game, type "npm run dev" into the terminal of your coding editor.

## Reset Game 
To reset the records of the game, go to the browser console in the inspector and paste "localStorage.clear();" into the console, then reload the page.
